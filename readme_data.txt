There are three data files for the project 1) small_file.txt, 2)test1.txt, 3 test.txt. Small_file is used to train the data it has around 12000 question and answer, test1.txt file contains the pasage , question related to the same passage and answer for validation. The test.txt is has a list of the different test cases. copy paste one passage and one question{everythin is in one line} from test data to test1.txt  and past it to the test1.txt file to obtained the output. Please do not include more then one passage.
The format of input and test file is of the format "PARAGRAPH +\t " +" question +\t" + "answer". So if the input line is split using "\t" then it will gnerate three list.
